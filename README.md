# Shammy

Shammy is an open-source personal data aggregation platform that unifies email, calendar, files, commits, and more into a single AI-ready datastore.

## Getting Started

```bash
pnpm install
./scripts/dev/start.sh
```

Services run through `infra/docker/docker-compose.yml`. Visit `http://localhost:3000` for the web app, `http://localhost:8001` for Airbyte, and `http://localhost:7700` for Meilisearch.
